# recently-files-tree

[English Intro](README.MD) | [中文介绍](README-ZH.md)

vscode自带的最近文件 是以打开文件顺序排序, 本拓展是以目录结构顺序展开, 更容易寻找

## 特点

- 以文件路径排序
- 支持收藏
- 支持过滤
